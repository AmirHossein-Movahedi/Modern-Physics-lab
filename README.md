# Modern Physics Lab ⚛️

This repository is dedicated to the **Modern Physics Lab** course, part of the Bachelor's degree program in Physics.

It contains:

* **Experimental Data:** Raw data collected from various lab experiments.
* **Data Analysis:** Scripts and notes used to process the collected data.
* **Results and Error Analysis:** The final calculated results, including thorough **error analysis** to ensure high accuracy.

---
**Goal:** The primary goal is to document the complete process—from conducting the experiment to accurately analyzing the data and determining the final, reliable result.
